# Buhay

`Buhay` is an implementation of [Particle Life](https://particle-life.com/) in [Rust](https://www.rust-lang.org/) with [Bevy](https://bevyengine.org/).

## Installation and Usage

To run the program, do:

```bash
cargo run --release
```
